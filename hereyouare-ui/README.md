# HereYouAre UI

React-приложение для загрузки изображений и просмотра результатов детекции **HereYouAre**:

## Пользовательский интерфейс

![Пример интерфейса HereYouAre UI](ui_example.png)

*Загрузка изображения, детекция объектов с рамками и миниатюрами найденных объектов.*

Загрузка → асинхронная детекция → опрос 4 раза в секунду → начальная рамка 30% по центру → поочерёдное появление угловых рамок (15% стороны) с анимацией.

## Стек

- **React 18** + **Vite 5**
- Запросы к API: `POST /v1/detect/async`, `GET /v1/job/{job_id}` (интервал 250 ms)

## Запуск

```bash
npm install
npm run dev
```

Приложение: http://localhost:5173

Сборка для продакшена: `npm run build`. Статика в `dist/`.

## Конфигурация

- **.env**: `VITE_HEREYOUARE_URL` — базовый URL бэкенда (по умолчанию `http://localhost:8082`).
- В **vite.config.js** настроен proxy для `/api` и `/v1` на бэкенд (при разработке можно задать `VITE_HEREYOUARE_URL=` и использовать относительные пути).

## Поведение

1. Пользователь загружает файл (drag-and-drop или выбор).
2. Нажимает «Запустить детекцию» → отправка на `POST /v1/detect/async`, получение `job_id`.
3. Опрос `GET /v1/job/{job_id}` каждые 250 ms.
4. При статусе `processing` отображается начальная рамка по центру (30% области).
5. При статусе `done` рамки по очереди «перелетают» к объектам (каждые 300 ms), рисуются угловые отрезки (15% стороны) в одном тоне.
